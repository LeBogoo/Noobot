# Noobot

## Requirements

-   MongoDB
-   Node.js ^16.9.0

## Installation

```bash
$ npm install
```

```bash
$ npm install -g ts-node
```

> Don't forget to create a .env file!
> Look at the .env_example file!

## Run

```bash
$ ts-node src/index.ts
```
